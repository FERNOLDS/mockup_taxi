ICONOS DE FUENTES DE DATOS
==========================

Coloca aquí los archivos de logo para cada integración.
El HTML los buscará por nombre exacto (PNG o SVG).

Archivos esperados:
  sii.png          → SII Chile
  mercadopago.png  → MercadoPago
  sumup.png        → SumUp
  tuu.png          → Tuu (Multicaja)
  bancoestado.png  → BancoEstado

Tamaño recomendado: 64×64 px o mayor (se escala a 32×32).
Fondo transparente recomendado.

Si el archivo no existe, el sistema intentará cargar el favicon
oficial de cada dominio como respaldo automático.
